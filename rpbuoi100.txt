5. Implement Trie (Prefix Tree) mỗi node trong Trie lưu:

Một mảng/Map các con (children)

Cờ isEnd để đánh dấu node kết thúc một từ

Khi insert duyệt từng ký tự tạo node nếu chưa có

Khi search duyệt từng ký tự nếu thiếu node thì false, nếu đủ và isEnd = true thì true

Khi startsWith duyệt từng ký tự nếu thiếu node →=thì false, nếu đủ thì true

-Tìm kiếm prefix nhanh hơn so với duyệt toàn bộ danh sách từ

Độ phức tạp O(L) cho mỗi thao tác (L = độ dài từ/prefix)

-Code:

class Trie {
    // Định nghĩa node trong Trie
    class TrieNode {
        TrieNode[] children; // mảng con cho 26 chữ cái
        boolean isEnd;       // đánh dấu kết thúc một từ
        
        TrieNode() {
            children = new TrieNode[26];
            isEnd = false;
        }
    }
    
    private TrieNode root;
    
    public Trie() {
        root = new TrieNode(); // khởi tạo Trie rỗng
    }
    
    // Thêm một từ vào Trie
    public void insert(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int index = c - 'a'; // chuyển ký tự thành index
            if (node.children[index] == null) {
                node.children[index] = new TrieNode(); // tạo node mới nếu chưa có
            }
            node = node.children[index];
        }
        node.isEnd = true; // đánh dấu kết thúc từ
    }
    
    // Kiểm tra từ có tồn tại trong Trie không
    public boolean search(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int index = c - 'a';
            if (node.children[index] == null) return false; // thiếu node thì không tồn tại
            node = node.children[index];
        }
        return node.isEnd; // chỉ true nếu đúng là kết thúc từ
    }
    
    // Kiểm tra có từ nào bắt đầu bằng prefix không
    public boolean startsWith(String prefix) {
        TrieNode node = root;
        for (char c : prefix.toCharArray()) {
            int index = c - 'a';
            if (node.children[index] == null) return false; // thiếu node thì không tồn tại prefix
            node = node.children[index];
        }
        return true; // duyệt hết prefix thành công thì tồn tại
    }
}


6.Kth Largest Element in a Stream dùng Min-Heap (PriorityQueue) để lưu k phần tử lớn nhất

Khi thêm số mới:

Thêm vào heap

Nếu heap có nhiều hơn k phần tử thì loại bỏ phần tử nhỏ nhất (root của min-heap)

Khi đó phần tử nhỏ nhất trong heap chính là k-th largest

-Heap cho phép thêm và loại bỏ phần tử trong O(log k)

Chỉ cần lưu k phần tử lớn nhất thay vì toàn bộ luồng để tiết kiệm bộ nhớ

-Code:

import java.util.PriorityQueue;

class KthLargest {
    private PriorityQueue<Integer> minHeap; // Min-Heap để lưu k phần tử lớn nhất
    private int k;
    
    public KthLargest(int k, int[] nums) {
        this.k = k;
        minHeap = new PriorityQueue<>(); // mặc định là min-heap
        
        // Thêm tất cả phần tử ban đầu vào heap
        for (int num : nums) {
            add(num);
        }
    }
    
    public int add(int val) {
        // Thêm phần tử mới vào heap
        minHeap.offer(val);
        
        // Nếu heap có nhiều hơn k phần tử thì loại bỏ phần tử nhỏ nhất
        if (minHeap.size() > k) {
            minHeap.poll();
        }
        
        // Root của heap chính là k-th largest
        return minHeap.peek();
    }
}

7.Last Stone Weight dùng Max-Heap (PriorityQueue với comparator đảo ngược) để luôn lấy ra 2 viên nặng nhất

Mỗi lần poll 2 viên tính kết quả nếu còn đá thì add lại vào heap

Lặp cho đến khi heap rỗng hoặc còn 1 viên

-Cho phép lấy nhanh 2 phần tử lớn nhất trong O(log n)

-Code:

import java.util.PriorityQueue;

class Solution {
    public int lastStoneWeight(int[] stones) {
        // Tạo max-heap (PriorityQueue với comparator đảo ngược)
        PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a);
        
        // Thêm tất cả viên đá vào heap
        for (int stone : stones) {
            maxHeap.offer(stone);
        }
        
        // Lặp cho đến khi còn <= 1 viên đá
        while (maxHeap.size() > 1) {
            int y = maxHeap.poll(); // viên nặng nhất
            int x = maxHeap.poll(); // viên nặng nhì
            
            if (y != x) {
                // Nếu khác nhau thì thêm viên mới có trọng lượng y - x
                maxHeap.offer(y - x);
            }
            // Nếu bằng nhau thì cả hai biến mất và không thêm gì
        }
        
        // Nếu còn 1 viên thì trả về trọng lượng nếu không thì trả về 0
        return maxHeap.isEmpty() ? 0 : maxHeap.peek();
    }
}
